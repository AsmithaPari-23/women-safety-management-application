# Women Safety Management Application

A full-stack web application designed to enhance women's safety by providing emergency assistance, secure authentication, and location-based safety features. The system enables users to quickly access help during emergencies through an intuitive interface.

## Features

- Secure user authentication
- SOS emergency alert system
- Live location sharing
- Emergency contact management
- Responsive and user-friendly interface

## Tech Stack

**Frontend:** React.js, HTML, CSS, JavaScript  
**Backend:** Node.js, Express.js  
**Database:** MongoDB

## Installation

```bash
git clone https://github.com/AsmithaPari-23/women-safety-management-application.git
cd women-safety-management-application
npm install
```

Configure the required environment variables and start the frontend and backend servers.
